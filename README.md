It is a Basic E Commerce Application Provodes Features like Creating or Uploading your own product, Editing Product Information, Removing the Product and Adding the Product into my Cart

To Run this project:-
1. install flask
2. install sqlite (if not already available)
3. run the main.py and paste the server URL in your Browser

The Project is Created on sql queries and backend logic mainly

Stack:-
1. Frontend -> HTML, CSS, Copilot
2. Backend -> Flask, Jinja
3. DataBase -> Sqlite3
